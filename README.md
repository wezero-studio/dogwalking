# Brittalent

Brittalent is a B2B appointment-setting agency landing page, built with modern web technologies to deliver a high-performance, conversion-optimized experience.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/` - Next.js App Router root layout and pages
- `src/components/sections/` - Major landing page sections (Hero, How It Works, Pricing, etc.)
- `src/components/ui/` - Reusable UI components from shadcn/ui
- `public/` - Static assets

## Deployment

This project is configured for easy deployment on [Vercel](https://vercel.com/). Connect your GitHub repository to Vercel to automatically deploy the `main` branch.
